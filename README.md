## Uruchomienie

1. Skopiuj projekt do katalogu serwera WWW (np. `/opt/lampp/htdocs/portfolio2`).
2. Upewnij się, że pliki w folderze `assets` są dostępne.
3. Otwórz `index.html` w przeglądarce lub uruchom lokalny serwer (np. XAMPP/LAMP).
